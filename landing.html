Attention has an exact line where its guarantees change character. We found it.

Sharpen attention's temperature from soft toward hard and, at one precise point, its routing crosses from provably certifiable to not, while the far side stays learnable. The bound that proves it holds for the IEEE float32 program a transformer actually runs, not an idealized model of it.

Temperature is the toggle. The drop is a proved separation, not an artifact: the routing stays measurable, and the hard side stays learnable across the line.

The proof is about the program that runs.

Almost every guarantee about a neural network is a guarantee about an idealized version of it: real numbers, exact arithmetic, infinite precision. The network that runs is none of those. It rounds at every step, in float32. This bound is stated about the executed operation, every term computable from the actual weights, and the certified model is proven equal, coordinate by coordinate, to the literal attention operation the code calls. It extends through depth to a full transformer stack, with attention, feed-forward, and layer normalization each bound to the literal operation it runs.

For a single attention head the rounding correction is derived all the way down to one floating-point atom. Through the full depth stack it is still supplied per layer. That last step is the remaining float32 node, and it is named, not hidden.

Two certificates, the same gap, a crossover.

A transformer router reads a discrete symbol from continuous attention scores. Its generalization gap is bounded two ways at once: a smooth certificate that is tighter when routing is soft, and a hard combinatorial certificate that is tighter when routing is sharp. They meet at a named crossover sharpness. Take the better of the two at every sharpness and the gap is controlled across the whole range.

Smooth wins when routing is soft, hard wins when it is sharp, and βStar is where they cross. Both bound the same routed-symbol gap.

Deeper and wider each strictly buy expressivity.

For the constrained routed model, adding one layer of depth strictly enlarges the routes the network can express, at every level, and so does adding one unit of width. These are proved strict separations: real routes that the shallower or narrower model provably cannot express, not gaps left open.

Under full self-attention, where the scores are quadratic, the picture splits. The cells become non-convex, yet the model stays learnable, because polynomial scores are finite-dimensional and so the class still has finite VC. The general-depth self-attention separation is the named open step.

Each added layer strictly enlarges the realizable routes; width does the same (arity n strictly inside arity n+1). Under self-attention the cells go non-convex but stay learnable, and the general-depth case is the open step.

Machine-checked, with nothing hidden.

Every result is verified in the Lean kernel with no use of sorry, and reduces to three standard logical axioms with nothing added. Where a step is still conditional, it is stated as a hypothesis rather than folded into a hidden budget. The frontier is drawn on the same page as the results.

Stated as open, on the same page

  • The single attention head's float32 rounding is derived to one atom; the depth stack still supplies its per-layer rounding.
  • Non-vacuity at depth is conditional on certified non-expansiveness; outside that regime the linear-in-depth bound does not apply.
  • The general-depth self-attention expressivity separation is open: the depth bound and the base rung are proved, an iterated-tent witness is the remaining step.
